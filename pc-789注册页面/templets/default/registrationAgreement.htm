789游戏中心用户协议

1. 概述

根据《网络游戏管理暂行规定》，文化部《网络游戏服务格式化协议必备条款》。

甲方为网络游戏运营企业，乙方为网络游戏用户。

1.1 本服务条款系由乙方与789游戏中心（甲方）提供的服务所达成的协议。甲方同意按本服务条款及甲方针对某项具体服务所制定的相关规定（简称甲方）为乙方提供服务。甲方针对某项具体服务所制定的相关规定包括但不限于789游戏中心服务条款及可能扩充的其他相应服务条款、游戏规则、公告等。

1.2 请于注册成为甲方所提供服务的用户前，详细阅读本服务条款所有内容，尤其是本服务条款中免除、限制789游戏中心责任或者保障用户权利的条款。

1.3 乙方须同意本服务条款的全部内容，并按照页面上的提示完成全部的注册程序。选择同意本服务条款，表明乙方自愿承诺接受本服务条款以及甲方相关规定的约束。如发生纠纷，乙方不得以未仔细阅读本服务条款为由实行抗辩。

2. 青少年保护

2.1 未成年人应经其监护人陪同阅读本服务条款并表示同意，方可接受本服务条款。监护人应加强对未成年人的监督和保护，因其未谨慎履行监护责任而损害未成年人利益或者影响789游戏中心利益的，应由监护人承担责任。

2.2 青少年用户应遵守全国青少年网络文明公约：要善于网上学习，不浏览不良信息；要诚实友好交流，不侮辱欺诈他人；要增强自护意识，不随意约会网友；要维护网络安全，不破坏网络秩序；要有益身心健康，不沉溺虚拟时空。

3. 用户账号注册

3.1 乙方承诺以其真实身份注册成为甲方的用户，并保证所提供的个人身份资料信息真实、完整、有效，依据法律规定和必备条款约定对所提供的信息承担相应的法律责任。

3.2 乙方以其真实身份注册成为甲方用户后，需要修改所提供的个人身份资料信息的，甲方应当及时、有效地为其提供该项服务。乙方可以选择789游戏中心的保护服务以保障账号安全。

3.3 个人资料发生变更，如乙方迟延更新个人注册信息导致影响甲方对该用户提供服务，则由此产生的全部责任均由乙方承担。

3.4 账号或昵称的注册须遵守公共秩序和善良风俗；不得有明确或者隐含非法、骚扰性、侮辱性、恐吓性、伤害性、破坏性、挑衅性、庸俗性、淫秽色情性等内容的文字以及其他侵犯他人合法利益和人格尊严的文字。

4. 用户账号密码的使用和保管

4.1 根据必备条款的约定，甲方有权审查乙方注册所提供的身份信息是否真实、有效，并应积极地采取技术与管理等合理措施保障用户账号的安全、有效；乙方应维持账号密码的安全，妥善保管，谨防被盗或泄露，不应将账号出借或者转让他人使用，并对任何人利用其账号所进行的活动负完全的责任。任何一方未尽上述义务导致账号密码遗失、账号被盗等情形而给乙方和他人的民事权利造成损害的，应当承担由此产生的法律责任。

4.2 甲方关于用户账号密码安全性的提示：

(A) 乙方设置密码要复杂，同时便于记忆。

(B) 乙方在公共场所上网时不要选择记住密码功能服务。

(C) 乙方可以定期更新密码，避免密码泄露。

(D) 乙方可以申请密码保护，在密码丢失或泄露时维护自己的权益。

(E) 乙方可以选择甲方在账号密码安全方面的特殊服务。

4.3 乙方发现其账号被他人非法使用，应及时根据甲方公布的处理方式通知甲方，并有权通知甲方采取措施暂停该账号的登录和使用。

4.4 甲方根据乙方的通知采取措施暂停乙方账号的登录和使用的，甲方应当要求乙方提供并核实与其注册身份信息相一致的个人有效身份信息。甲方核实乙方所提供的个人有效身份信息与所注册的身份信息相一致的，应当及时采取措施暂停乙方账号的登录和使用。

4.5 甲方违反4.4 款项的约定，未及时采取措施暂停乙方账号的登录和使用，因此而给乙方造成损失的，应当承担其相应的法律责任。

4.6 乙方没有提供其个人有效身份证件或者乙方提供的个人有效身份证件与所注册的身份信息不一致的，甲方有权拒绝乙方上述请求。

4.7 乙方为了维护其合法权益，向甲方提供与所注册的身份信息相一致的个人有效身份信息时，甲方应当为乙方提供账号注册人证明、原始注册信息等必要的协助和支持，并根据需要向有关行政机关和司法机关提供相关证据信息资料。

4.8 乙方对登录后所持账号产生的行为依法享有权利和承担责任。

4.9 乙方可以更改账号的密码，也可以申请新账号，但不可以要求删除旧账号。乙方向789游戏中心查询、修改或重置其账号密码时，必须提供完全准确的个人注册信息（包括注册时间），否则甲方有权本着为用户保密的原则不予告知。

5. 会员资格的规定

5.1 乙方可以通过实物卡充值等甲方规定的方式取得会员资格，享受甲方按照其规定提供的会员服务。

5.2 乙方在注册或兑换会员资格时，应谨慎核对其注册或兑换会员资格的账号。如非甲方原因导致其账号未获得会员资格，甲方对由此可能导致的用户利益损失不承担任何责任。

5.3 为保障乙方正常行使其基于会员资格享有的权利，乙方需申请密码保护，并提供真实准确的个人信息。

5.4 兑换的会员资格仅限于乙方固定账号使用，不得将其转到其他账号上。

6.用户权利

6.1 一经账号注册完成，乙方即享有该账号的使用权。

6.2 乙方有权通过甲方公布的客户服务部门的联系方式咨询相关问题。

6.3 乙方有权参加甲方组织的相关活动，并按照活动规则享有相关权益。

6.4 乙方有权向甲方提出合理化意见和建议，有权对甲方的服务进行善意的监督。

6.5 拥有会员资格的乙方享有会员权益。

7.1 乙方应遵守中华人民共和国相关法律、法规、规章以及其他规范性文件的要求；遵守与网络服务有关的网络协议、规定和程序；遵守本服务条款以及甲方相关规定；不从事任何损害国家、甲方以及其他用户的合法利益的行为。

7.6 乙方在一个月内赠送、转移至其他帐号积分总和不得超过该帐号最近二个月充值获得积分总和，乙方不得利用任何方式直接或变相的将在游戏中获胜而获得的游戏积分转移，不得从事作弊等损害游戏公平性的行为，并承诺接受甲方对其游戏数据进行分析，接受甲方对积分转移判定结果。

7.8 乙方承诺在发现甲方可能存在系统漏洞时，须及时告知甲方，并不得从事任何利用系统漏洞的行为。
